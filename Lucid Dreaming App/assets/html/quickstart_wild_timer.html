WILD Timer

W.I.L.D. stands for Wake Induced Lucid Dreaming. This technique involves going into a dream straight from the waking state.
Here's a comprehensive tutorial . Lucid Dreaming App is a good aid in keeping your mind awake long enough to enter dream consciously. There are 2 main methods of WILD
  • WILD before REM: involves waking up 4-5 hours after going to bed and staying awake in bed for 40-50 minutes
  • To use this method select Preferences: Configure WILD Timers and pick "WILDTimerConfig.txt.gzip". This file includes reminders spread over 50 minutes
  • Dream Reentry or DEILD is a more advanced technique
  • Dream reentry takes place within a couple minutes after waking up from a dream
  • To use this method select Preferences: Configure WILD Timers and pick "WILDTimerReentryConfig.txt.gzip". This file spreads reminders over 16 minutes
WILD Timer is a regular set of timers that go off exactly when configured.
